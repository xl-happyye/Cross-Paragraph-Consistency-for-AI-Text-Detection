## Dataset Description

The dataset consists of two main components: **human-written texts**, **AI-generated texts**, and **adversarially perturbed AI samples**. It is designed for evaluating AI-generated text detection and robustness under attack settings.

---

### 1. `human&_model-area.xlsx`

This file contains paired human-written and AI-generated texts across multiple domains (“areas”).

- Each area includes:
  - **1,600 human-written texts**
  - **1,600 AI-generated texts**
- AI texts are generated by corresponding large language models based on the same or similar prompts as human-written texts.
- The dataset is balanced between human and machine-generated samples.
- Samples are organized by semantic topic or prompt category within each area to ensure comparability.

---

### 2. `area_attack.xlsx`

This file contains adversarially perturbed AI-generated texts for robustness evaluation.

- Each area includes:
  - **300 human-written reference texts**
  - **300 perturbed Claude-generated AI texts**
- The perturbed texts are created by applying attack strategies to the original AI outputs while preserving semantic meaning.
- This dataset is used to evaluate the robustness of AI text detection systems under adversarial conditions.
