## Dataset Description

This dataset is constructed based on the DNA-DetectLLM benchmark and further processed for cross-paragraph consistency analysis. It consists of human-written texts and AI-generated texts generated by different large language models.

### `human&_model-area.xlsx`

This file contains paired human-written and AI-generated texts across three domains:

- `XSum` → news
- `WritingPrompt` → story
- `Arxiv` → academic

In the dataset files, the `area` field corresponds to these domains (`news`, `story`, and `academic`).

AI-generated texts are produced by:
- `GPT-4`
- `Gemini`
- `Claude`

Each domain includes:
- 1,600 human-written texts
- 1,600 AI-generated texts for each model

Additional details:
- AI-generated texts are generated using the same or semantically similar prompts as the human-written texts.
- The dataset is balanced between human and AI-generated samples.
- Samples are reorganized to support discourse-level and cross-paragraph consistency analysis.
